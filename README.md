## Installation
***
A little intro about the installation. 
```
$ git clone https://github.com/EdgarJoel123/ProyectoConcurso.git
$ cd ../path/to/the/file
$ npm install
$ npm install -g json-server
$ npm install -g json-server
$ json-server base.son
$ ng serve

```
